      Overview

      Volatile adds the attr_volatile class method when included.

      foo = Foo.new
foo.bar
#=> 1
foo.bar = 2
#=> 2

      Examples:

      class Foo
  include Concurrent::Synchronization::Volatile

  attr_volatile :bar

  def initialize
    self.bar = 1
  end
end
