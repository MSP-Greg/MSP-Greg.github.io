      Overview

      Contains the result of a successful invocation of the following actions:

      • CreateDBSnapshot

      • DeleteDBSnapshot

      This data type is used as a response element in the DescribeDBSnapshots action.

      Returned by:

      • CopyDBSnapshotResult#db_snapshot
      • CreateDBSnapshotResult#db_snapshot
      • Aws::RDS::Types::DeleteDBSnapshotResult#db_snapshot
