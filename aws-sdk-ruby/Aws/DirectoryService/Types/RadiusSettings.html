      Overview

      Contains information about a Remote Authentication Dial In User Service (RADIUS) server.

      Returned by:

      • DirectoryDescription#radius_settings
      • EnableRadiusRequest#radius_settings
      • UpdateRadiusRequest#radius_settings

      Note: When passing RadiusSettings as input to a Client method, you can use a vanilla Hash:

      {
  radius_servers: ["Server"],
  radius_port: 1,
  radius_timeout: 1,
  radius_retries: 1,
  shared_secret: "RadiusSharedSecret",
  authentication_protocol: "PAP", # accepts PAP, CHAP, MS-CHAPv1, MS-CHAPv2
  display_label: "RadiusDisplayLabel",
  use_same_username: false,
}
