      Overview

      Describes the launch specification for an instance.

      Returned by:

      • Aws::EC2::Types::RequestSpotInstancesRequest#launch_specification

      Note: When passing RequestSpotLaunchSpecification as input to a Client method, you can use a vanilla Hash:
