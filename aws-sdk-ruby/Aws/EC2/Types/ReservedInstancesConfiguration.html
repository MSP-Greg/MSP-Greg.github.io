      Overview

      Describes the configuration settings for the modified Reserved Instances.

      Returned by:

      • Aws::EC2::Types::ReservedInstancesModificationResult#target_configuration

      Note: When passing ReservedInstancesConfiguration as input to a Client method, you can use a vanilla Hash:
