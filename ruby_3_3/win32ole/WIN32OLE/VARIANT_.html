      Overview

      The VARIANT module includes constants of VARIANT type constants. The constants is used when creating ::WIN32OLE_VARIANT object.

      obj = WIN32OLE_VARIANT.new("2e3", WIN32OLE::VARIANT::VT_R4)
obj.value # => 2000.0
