    #|(object)  ⇒ true   
      

      
  
      
    

      Returns true:
      


      true | Object.new # => true
true | false      # => true
true | nil        # => true
      


      Argument object is evaluated. This is different from true with the short-circuit operator, whose operand is evaluated only if necessary:
      


      true | raise # => Raises RuntimeError.
true || raise # => true
