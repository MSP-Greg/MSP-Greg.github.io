      Overview

      Note:

      On Mac OS X, file watching only works when kqueue is enabled

      Utility class that is useful for file monitoring. Supported events are

      • File is modified
      • File is deleted
      • File is moved

def receive_data(data)
  case data
  when Cmodified
    file_modified
  when Cdeleted
    file_deleted
  when Cmoved
    file_moved
  end
end

      
  
      

      

      


      
  
      
    #stop_watching  
  
      

      
  
      
    
      Discontinue monitoring of the file.
      


      This involves cleaning up the underlying monitoring details with kqueue/inotify, and in turn firing Connection#unbind.
This will be called automatically when a file is deleted. User code may call it as well.
