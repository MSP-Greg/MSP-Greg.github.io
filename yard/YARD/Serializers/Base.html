      Overview

      This class is abstract.

      Override this class to implement a custom serializer.

      The abstract base serializer. ::YARD::Serializers allow templates to be rendered to various endpoints. For instance, a FileSystemSerializer would allow template contents to be written to the filesystem

      To implement a custom serializer, override the following methods:

      • #serialize
      • #serialized_path

      Optionally, a serializer can implement before and after filters:

      • #before_serialize
      • #after_serialize
