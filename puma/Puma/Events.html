      Class: Puma::Events

      Overview

      This is an event sink used by Server to handle lifecycle events such as :after_booted, :before_restart, and :after_stopped. Using DSL it is possible to register callback hooks for each event type.

def register(hook, obj=nil, &blk)
  if obj and blk
    raise "Specify either an object or a block, not both"
  end

  h = obj || blk

  @hooks[hook] << h

  h
end
