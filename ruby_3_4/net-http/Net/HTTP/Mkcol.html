      Overview

      Class for representing WebDAV method MKCOL:

      require 'net/http'
uri = URI('http://example.com')
hostname = uri.hostname # => "example.com"
req = Net::HTTP::Mkcol.new(uri) # => #<Net::HTTP::Mkcol MKCOL>
res = Net::HTTP.start(hostname) do |http|

      http.request(req)

      end
