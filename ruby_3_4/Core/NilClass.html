      Overview

      The class of the singleton object nil.

      Several of its methods act as operators:

      • #&

      • #|

      • #===

      • #=~

      • #^

      Others act as converters, carrying the concept of nullity to other classes:

      • #rationalize

      • #to_a

      • #to_c

      • #to_h

      • #to_r

      • #to_s

      While nil doesnâ€™t have an explicitly defined #to_hash method, it can be used in ** unpacking, not adding any keyword arguments.

      Another method provides inspection:

      • #inspect

      Finally, there is this query method:

      • #nil?
