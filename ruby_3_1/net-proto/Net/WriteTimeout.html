      Overview

      WriteTimeout, a subclass of Timeout::Error, is raised if a chunk of the response cannot be written within the write_timeout. Not raised on Windows.

def message
  msg = super
  if @io
    msg = "#{msg} with #{@io.inspect}"
  end
  msg
end
