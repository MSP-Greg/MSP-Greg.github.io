      Overview

      Active Job adapters

      Active Job has adapters for the following queueing backends:

      • Backburner[github.com/nesquena/backburner]

      • Delayed[github.com/collectiveidea/delayed_job]

      • Qu[github.com/bkeepers/qu]

      • Que[github.com/chanks/que]

      • queue_classic[github.com/QueueClassic/queue_classic]

      • Resque[github.com/resque/resque/tree/1-x-stable]

      • Sidekiq[sidekiq.org]

      • Sneakers[github.com/jondot/sneakers]

      • Sucker[github.com/brandonhilkert/sucker_punch]

      Backends Features

      |                   | Async | Queues | Delayed    | Priorities | Timeout | Retries |
|-------------------|-------|--------|------------|------------|---------|---------|
| Backburner        | Yes   | Yes    | Yes        | Yes        | Job     | Global  |
| Delayed Job       | Yes   | Yes    | Yes        | Job        | Global  | Global  |
| Qu                | Yes   | Yes    | No         | No         | No      | Global  |
| Que               | Yes   | Yes    | Yes        | Job        | No      | Job     |
| queue_classic     | Yes   | Yes    | Yes*       | No         | No      | No      |
| Resque            | Yes   | Yes    | Yes (Gem)  | Queue      | Global  | Yes     |
| Sidekiq           | Yes   | Yes    | Yes        | Queue      | No      | Job     |
| Sneakers          | Yes   | Yes    | No         | Queue      | Queue   | No      |
| Sucker Punch      | Yes   | Yes    | No         | No         | No      | No      |
| Active Job Inline | No    | Yes    | N/A        | N/A        | N/A     | N/A     |

      NOTE: queue_classic supports job scheduling since version 3.1. For older versions you can use the queue_classic-later gem.
