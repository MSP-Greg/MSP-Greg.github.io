      Instance Method Details

      #create_pull_review(repo, pull_number, options = {}) ⇒ Sawyer::Resource

      Create a pull request review

      Parameters:

      • repo (Integer, String, Repository, Hash) —

        A GitHub repository

      • pull_number (Integer) —

        The number of the pull

      • options (Hash) (defaults to: {}) —

        a customizable set of options

      Options Hash (options):

      • :commit_id (String) —

        The ID of the commit

      • :body (String) —

        Required when using REQUEST_CHANGES or COMMENT for the event parameter. The body text of the pull request review.

      • :event (String) —

        The review action you want to perform. The review actions include: APPROVE, REQUEST_CHANGES, or COMMENT. By leaving this blank, you set the review action state to PENDING, which means you will need to submit the pull request review (https://developer.github.com/v3/pulls/reviews/#submit-a-pull-request-review) when you are ready.

      • :comments (Array) —

        Use the following table to specify the location, destination, and contents of the draft review comment.

      Returns:

      • (Sawyer::Resource) —

        The new review

      See Also:

      • https://developer.github.com/v3/pulls/reviews/#create-a-pull-request-review

def dismiss_pull_review(repo, pull_number, review_id, message, options = {})
  opts = options.dup
  opts[:message] = message
  put "#{Repository.path repo}/pulls/#{pull_number}/reviews/#{review_id}/dismissals", opts

def submit_pull_review(repo, pull_number, review_id, event, options = {})
  opts = options.dup
  opts[:event] = event.to_s.downcase
  post "#{Repository.path repo}/pulls/#{pull_number}/reviews/#{review_id}/events", opts
end

def update_pull_review(repo, pull_number, review_id, body, options = {})
  opts = options.dup
  opts[:body] = body
  put "#{Repository.path repo}/pulls/#{pull_number}/reviews/#{review_id}", opts
end
