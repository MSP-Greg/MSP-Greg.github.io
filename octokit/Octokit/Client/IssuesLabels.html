      #add_issue_labels(repo, issue_number, labels, options = {}) ⇒ Sawyer::Resource

      Add labels to an issue

      Parameters:

      • repo (Integer, String, Repository, Hash) —

        A GitHub repository

      • issue_number (Integer) —

        The number of the issue

      • labels (Array) —

        The name of the label to add to the issue. Must contain at least one label. Note: Alternatively, you can pass a single label as a string or an array of labels directly, but GitHub recommends passing an object with the labels key.

      Returns:

      • (Sawyer::Resource) —

        The list of new labels

def add_issue_labels(repo, issue_number, labels, options = {})
  opts = options.dup
  opts[:labels] = labels
  post "#{Repository.path repo}/issues/#{issue_number}/labels", opts
end

def create_issue_label(repo, name, color, options = {})
  opts = options.dup
  opts[:name] = name
  opts[:color] = color
  post "#{Repository.path repo}/labels", opts
