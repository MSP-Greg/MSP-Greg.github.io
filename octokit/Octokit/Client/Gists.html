      #create_gist(files, options = {}) ⇒ Sawyer::Resource

      Create a gist

      Parameters:

      • files (Object) —

        The filenames and content of each file in the gist. The keys in the files object represent the filename and have the type string.

      • options (Hash) (defaults to: {}) —

        a customizable set of options

      Options Hash (options):

      • :description (String) —

        A descriptive name for this gist.

      • :public (Boolean) —

        When true, the gist will be public and available for anyone to see.

      Returns:

      • (Sawyer::Resource) —

        The new gist

      See Also:

      • https://developer.github.com/v3/gists/#create-a-gist

def create_gist(files, options = {})
  opts = options.dup
  opts[:files] = files
  post 'gists', opts
end
