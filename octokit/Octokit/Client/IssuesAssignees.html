      #add_issue_assignees(repo, issue_number, options = {}) ⇒ Sawyer::Resource

      Add assignees to an issue

      Parameters:

      • repo (Integer, String, Repository, Hash) —

        A GitHub repository

      • issue_number (Integer) —

        The number of the issue

      • options (Hash) (defaults to: {}) —

        a customizable set of options

      Options Hash (options):

      • :assignees (Array) —

        Usernames of people to assign this issue to. NOTE: Only users with push access can add assignees to an issue. Assignees are silently ignored otherwise.

      Returns:

      • (Sawyer::Resource) —

        The updated issue

def add_issue_assignees(repo, issue_number, options = {})
  post "#{Repository.path repo}/issues/#{issue_number}/assignees", options
end

  boolean_from_response :get, "#{Repository.path repo}/assignees/#{assignee}", options
end
