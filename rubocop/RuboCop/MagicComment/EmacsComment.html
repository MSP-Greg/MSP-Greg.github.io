      Overview

      Wrapper for Emacs style magic comments.

      Examples:

      Emacs style comment

      comment = RuboCop::MagicComment.parse(
  '# -*- encoding: ASCII-8BIT -*-'
)

comment.encoding # => 'ascii-8bit'

      • https://www.gnu.org/software/emacs/manual/html_node/emacs/Specify-Coding.html
