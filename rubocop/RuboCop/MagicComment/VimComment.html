      Overview

      Wrapper for Vim style magic comments.

      Examples:

      Vim style comment

      comment = RuboCop::MagicComment.parse(
  '# vim: filetype=ruby, fileencoding=ascii-8bit'
)

comment.encoding # => 'ascii-8bit'
