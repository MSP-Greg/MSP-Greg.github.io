      Instance Attribute Details

      #allow_users_to_change_password ⇒ Boolean (rw)

      Allows all ::Aws::IAM users in your account to use the AWS Management Console to change their own passwords. For more information, see Letting ::Aws::IAM users change their own passwords in the IAM User Guide.

      If you do not specify a value for this parameter, then the operation uses the default value of false. The result is that ::Aws::IAM users in the account do not automatically have permissions to change their own password.

    #hard_expiry  ⇒ Boolean  (rw)
  
      

      
  
      
    
      Prevents IAM users from setting a new password after their password
has expired. The IAM user cannot be accessed until an administrator
resets the password.
      

      If you do not specify a value for this parameter, then the operation
uses the default value of false. The result is that ::Aws::IAM users can
change their passwords after they expire and continue to sign in as
the user.
