      Overview

      This class provides a resource oriented interface for ::Aws::IAM. To create a resource object:

      resource = Aws::IAM::Resource.new(region: 'us-west-2')

      You can supply a client object with custom configuration that will be used for all resource operations. If you do not pass :client, a default client will be constructed.

      client = Aws::IAM::Client.new(region: 'us-west-2')
resource = Aws::IAM::Resource.new(client: client)

of these characters are valid in a password. However, many tools, such
as the AWS Management Console, might restrict the ability to type
certain characters because they have special meaning within that tool.

following non-alphanumeric characters:
        

        ! @ # $ % ^ & * ( ) _ + - = [ ] \{ \} | '
        

        If you do not specify a value for this parameter, then the operation
uses the default value of false. The result is that passwords do not
require at least one symbol character.
        

                
        • 
        
      •

          (Boolean)
            — 
        Specifies whether ::Aws::IAM user passwords must contain at least one numeric
character (0 to 9).
        

        If you do not specify a value for this parameter, then the operation

require at least one numeric character.
        

                
        • 
        
      • 
          :require_uppercase_characters
          (Boolean)
            — 
        Specifies whether ::Aws::IAM user passwords must contain at least one
uppercase character from the ISO basic Latin alphabet (A to Z).
        

        If you do not specify a value for this parameter, then the operation
uses the default value of false. The result is that passwords do not
require at least one uppercase character.
        

                
        • 
        
      • 
          :require_lowercase_characters
          (Boolean)
            — 
        Specifies whether ::Aws::IAM user passwords must contain at least one
lowercase character from the ISO basic Latin alphabet (a to z).
        

        If you do not specify a value for this parameter, then the operation
uses the default value of false. The result is that passwords do not
require at least one lowercase character.
        

                
        • 
        
      •

          (Boolean)
            — 
        Allows all ::Aws::IAM users in your account to use the AWS Management Console

users change their own passwords in the IAM User Guide.
        

        If you do not specify a value for this parameter, then the operation
uses the default value of false. The result is that ::Aws::IAM users in the

        • 
        
      •

          (Integer)

        The number of days that an ::Aws::IAM user password is valid.
        

        If you do not specify a value for this parameter, then the operation

        • 
        
      •

          (Boolean)
            —

        If you do not specify a value for this parameter, then the operation
uses the default value of false. The result is that ::Aws::IAM users can
change their passwords after they expire and continue to sign in as
the user.
