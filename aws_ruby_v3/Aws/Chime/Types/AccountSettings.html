      Overview

      Note:

      When making an API call, you may pass AccountSettings data as a hash:

      {
  disable_remote_control: false,
  enable_dial_out: false,
}

      Settings related to the Amazon Chime account. This includes settings that start or stop remote control of shared screens, or start or stop the dial-out option in the Amazon Chime web application. For more information about these settings, see Use the Policies Page in the Amazon ::Aws::Chime Administration Guide.
