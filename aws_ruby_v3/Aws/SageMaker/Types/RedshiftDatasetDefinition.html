      Instance Attribute Summary

      • #cluster_id ⇒ String rw

        The Redshift cluster Identifier.

      • #cluster_role_arn ⇒ String rw

        The IAM role attached to your ::Aws::Redshift cluster that Amazon SageMaker uses to generate datasets.

      • #database ⇒ String rw

        The name of the ::Aws::Redshift database used in ::Aws::Redshift query execution.

      • #db_user ⇒ String rw

        The database user name used in ::Aws::Redshift query execution.

      • #kms_key_id ⇒ String rw

        The AWS Key Management Service (AWS ::Aws::KMS) key that Amazon SageMaker uses to encrypt data from a ::Aws::Redshift execution.

      • #output_compression ⇒ String rw

        The compression used for ::Aws::Redshift query results.

      • #output_format ⇒ String rw

        The data storage format for ::Aws::Redshift query results.

      • #output_s3_uri ⇒ String rw

        The location in Amazon S3 where the ::Aws::Redshift query results are stored.

      • #query_string ⇒ String rw

        The SQL query statements to be executed.
