        The threshold for the class probability to be evaluated as a positive result.

        Whether input data distributed in Amazon S3 is fully replicated or sharded by an ::Aws::S3 key.

      • #s3_input_mode ⇒ String rw

        Whether the Pipe or File is used as the input mode for transfering data for the monitoring job.

      • #start_time_offset ⇒ String rw
