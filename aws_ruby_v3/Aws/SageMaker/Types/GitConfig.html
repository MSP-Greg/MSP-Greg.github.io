      Overview

      Note:

      When making an API call, you may pass GitConfig data as a hash:

      {
  repository_url: "GitConfigUrl", # required
  branch: "Branch",
  secret_arn: "SecretArn",
}

      Specifies configuration details for a Git repository in your AWS account.
