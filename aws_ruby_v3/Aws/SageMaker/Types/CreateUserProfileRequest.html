SingleSignOnUserValue. Currently, the only supported value is
"UserName". If the Domain's AuthMode is ::Aws::SSO, this field is
required. If the Domain's AuthMode is not ::Aws::SSO, this field cannot be
specified.

UserProfile. If the Domain's AuthMode is ::Aws::SSO, this field is
required, and must match a valid username of a user in your
