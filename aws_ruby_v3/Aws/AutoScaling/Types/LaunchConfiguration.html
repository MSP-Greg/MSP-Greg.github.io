    #user_data  ⇒ String  (rw)
  
      

      
  
      
    
      The user data to make available to the launched ::Aws::EC2 instances. For
more information, see Instance metadata and user data (Linux)
and Instance metadata and user data (Windows). If you are using
a command line tool, base64-encoding is performed for you, and you
can load the text from a file. Otherwise, you must provide
base64-encoded text. User data is limited to 16 KB.
