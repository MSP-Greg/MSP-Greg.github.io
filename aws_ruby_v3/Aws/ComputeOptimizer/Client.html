      Overview

      An API client for ::Aws::ComputeOptimizer. To construct a client, you need to configure a :region and :credentials.

      client = Aws::ComputeOptimizer::Client.new(
  region: region_name,
  credentials: credentials,
  # ...
)

      For details on configuring region and credentials see the developer guide.

      See #initialize for a full list of supported configuration options.

      Exports optimization recommendations for Auto Scaling groups.
      

      Recommendations are exported in a comma-separated values (.csv) file,
and its metadata in a JavaScript Object Notation (.json) file, to an
existing Amazon Simple Storage Service (Amazon ::Aws::S3) bucket that you
specify. For more information, see Exporting Recommendations in
the Compute Optimizer User Guide.
      

      You can have only one Auto Scaling group export job in progress per
AWS Region.
