      Instance Attribute Details

      #capabilities ⇒ Array<String> (rw)

      The capabilities found within the template. If your template contains ::Aws::IAM resources, you must specify the CAPABILITY_IAM or CAPABILITY_NAMED_IAM value for this parameter when you use the CreateStack or UpdateStack actions with your template; otherwise, those actions return an InsufficientCapabilities error.

      For more information, see Acknowledging ::Aws::IAM Resources in AWS ::Aws::CloudFormation Templates.
