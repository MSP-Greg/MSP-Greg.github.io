    #cluster_discovery_endpoint  ⇒ Types::Endpoint  (rw)
  
      

      
  
      
    
      The configuration endpoint for this ::Aws::DAX cluster, consisting of a DNS
name and a port number. ::Aws::DAX::Client applications can specify this
endpoint, rather than an individual node endpoint, and allow the ::Aws::DAX
client software to intelligently route requests and responses to
nodes in the ::Aws::DAX cluster.
