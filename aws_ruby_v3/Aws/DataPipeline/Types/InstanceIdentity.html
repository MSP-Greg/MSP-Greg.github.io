      Overview

      Note:

      When making an API call, you may pass InstanceIdentity data as a hash:

      {
  document: "string",
  signature: "string",
}

      Identity information for the ::Aws::EC2 instance that is hosting the task runner. You can get this value by calling a metadata URI from the ::Aws::EC2 instance. For more information, see Instance Metadata in the Amazon Elastic Compute Cloud User Guide. Passing in this value proves that your task runner is running on an ::Aws::EC2 instance, and ensures the proper AWS Data Pipeline service charges are applied to your pipeline.
