      Overview

      When SMS returns an error response, the Ruby SDK constructs and raises an error. These errors all extend Aws::SMS::Errors::ServiceError < ::Aws::Errors::ServiceError

      You can rescue all ::Aws::SMS errors using ServiceError:

      begin
  # do stuff
rescue Aws::SMS::Errors::ServiceError
  # rescues all SMS API errors
end

      Request Context

      ServiceError objects have a #context method that returns information about the request that generated the error. See ::Seahorse::Client::RequestContext for more information.
