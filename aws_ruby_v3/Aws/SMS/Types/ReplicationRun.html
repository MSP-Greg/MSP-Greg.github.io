    #kms_key_id  ⇒ String  (rw)
  
      

      
  
      
    
      The ID of the ::Aws::KMS key for replication jobs that produce encrypted
AMIs. This value can be any of the following:
      

        

      • 

        ::Aws::KMS key ID
        

        • 

      • 

        ::Aws::KMS key alias
        

        • 

      • 

        ::Aws::ARN referring to the ::Aws::KMS key ID
        

        • 

      • 

        ::Aws::ARN referring to the ::Aws::KMS key alias
        

        • 

      

      If encrypted is true but a ::Aws::KMS key ID is not specified, the
customer's default ::Aws::KMS key for Amazon EBS is used.
