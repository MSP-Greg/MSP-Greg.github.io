      Instance Attribute Details

      #ad_decision_server_url ⇒ String (rw)

      The URL for the ad decision server (ADS). This includes the specification of static parameters and placeholders for dynamic parameters. AWS Elemental ::Aws::MediaTailor substitutes player-specific and session-specific parameters as needed when calling the ADS. Alternately, for testing you can provide a static VAST URL. The maximum length is 25,000 characters.

    #slate_ad_url  ⇒ String  (rw)
  
      

      
  
      
    
      The URL for a high-quality video asset to transcode and use to fill
in time that's not used by ads. AWS Elemental ::Aws::MediaTailor shows the
slate to fill in gaps in media content. Configuring the slate is
optional for non-VPAID configurations. For VPAID, the slate is
required because ::Aws::MediaTailor provides it in the slots that are
designated for dynamic ad content. The slate must be a high-quality
asset that contains both audio and video.
