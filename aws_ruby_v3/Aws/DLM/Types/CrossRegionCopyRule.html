      Note:

      When making an API call, you may pass CrossRegionCopyRule data as a hash:

      {
  target_region: "TargetRegion",
  target: "Target",
  encrypted: false, # required
  cmk_arn: "CmkArn",
  copy_tags: false,
  retain_rule: {
    interval: 1,
    interval_unit: "DAYS", # accepts DAYS, WEEKS, MONTHS, YEARS
