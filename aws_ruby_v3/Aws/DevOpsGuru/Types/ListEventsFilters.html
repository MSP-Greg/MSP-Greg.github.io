      Instance Attribute Summary

      • #data_source ⇒ String rw

        The source, AWS_CLOUD_TRAIL or AWS_CODE_DEPLOY, of the events you want returned.

      • #event_class ⇒ String rw

        The class of the events you want to filter for, such as an infrastructure change, a deployment, or a schema change.

      • #event_source ⇒ String rw

        The AWS source that emitted the events you want to filter for.

      • #event_time_range ⇒ Types::EventTimeRange rw

        A time range during which you want the filtered events to have occurred.

      • #insight_id ⇒ String rw

        An ID of an insight that is related to the events you want to filter for.

      • #resource_collection ⇒ Types::ResourceCollection rw

        A collection of AWS resources supported by DevOps Guru.
