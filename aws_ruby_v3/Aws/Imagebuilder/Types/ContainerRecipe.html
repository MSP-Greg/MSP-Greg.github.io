        The date when this container recipe was created.

      • #encrypted ⇒ Boolean rw

        A flag that indicates if the target container is encrypted.

      • #kms_key_id ⇒ String rw

        Identifies which ::Aws::KMS key is used to encrypt the container image for distribution to the target Region.

      • #name ⇒ String rw

        The name of the container recipe.

      • #owner ⇒ String rw

        The owner of the container recipe.

      • #parent_image ⇒ String rw

        The source image for the container recipe.

      • #platform ⇒ String rw

        The system platform for the container, such as Windows or Linux.

      • #tags ⇒ Hash<String,String> rw

        Tags that are attached to the container recipe.

      • #target_repository ⇒ Types::TargetContainerRepository rw

        The destination repository for the container image.

      • #version ⇒ String rw

        The semantic version of the container recipe (<major>.<minor>.<patch>).

      • #working_directory ⇒ String rw

        The working directory for use during build and test workflows.
