      Overview

      Note:

      When making an API call, you may pass JobReport data as a hash:

      {
  bucket: "S3BucketArnString",
  format: "Report_CSV_20180820", # accepts Report_CSV_20180820
  enabled: false, # required
  prefix: "ReportPrefixString",
  report_scope: "AllTasks", # accepts AllTasks, FailedTasksOnly
}

      Contains the configuration parameters for a job-completion report.

      Instance Attribute Summary

      • #bucket ⇒ String rw

        The Amazon ::Aws::S3Control::Resource Name (ARN) for the bucket where specified job-completion report will be stored.

      • #enabled ⇒ Boolean rw

        Indicates whether the specified job will generate a job-completion report.

      • #format ⇒ String rw

        The format of the specified job-completion report.

      • #prefix ⇒ String rw

        An optional prefix to describe where in the specified bucket the job-completion report will be stored.

      • #report_scope ⇒ String rw

        Indicates whether the job-completion report will include details of all tasks or only failed tasks.
