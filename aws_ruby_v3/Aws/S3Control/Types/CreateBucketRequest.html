      • #grant_read ⇒ String rw

        Allows grantee to list the objects in the bucket.

      • #grant_read_acp ⇒ String rw

        Specifies whether you want ::Aws::S3 Object Lock to be enabled for the new bucket.

      • #outpost_id ⇒ String rw

        The ID of the ::Aws::Outposts where the bucket is being created.
