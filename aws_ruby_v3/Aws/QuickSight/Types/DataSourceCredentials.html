      #copy_source_arn ⇒ String (rw)

      The Amazon ::Aws::QuickSight::Resource Name (ARN) of a data source that has the credential pair that you want to use. When CopySourceArn is not null, the credential pair from the data source in the ::Aws::ARN is used as the credentials for the DataSourceCredentials structure.
