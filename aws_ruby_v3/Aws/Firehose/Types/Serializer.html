      The serializer that you want ::Aws::Kinesis Data ::Aws::Firehose to use to convert data to the target format before writing it to Amazon S3. ::Aws::Kinesis Data ::Aws::Firehose supports two types of serializers: the ORC SerDe and the Parquet SerDe.
