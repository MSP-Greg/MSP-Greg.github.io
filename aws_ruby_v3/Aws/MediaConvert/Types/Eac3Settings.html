      Instance Attribute Summary

      • #attenuation_control ⇒ String rw

        If set to ATTENUATE_3_DB, applies a 3 dB attenuation to the surround channels.

      • #bitrate ⇒ Integer rw

        Specify the average bitrate in bits per second.

      • #bitstream_mode ⇒ String rw

        Specify the bitstream mode for the E-AC-3 stream that the encoder emits.

      • #coding_mode ⇒ String rw

        Dolby Digital Plus coding mode.

      • #dc_filter ⇒ String rw

        Activates a DC highpass filter for all input channels.

      • #dialnorm ⇒ Integer rw

        Sets the dialnorm for the output.

      • #dynamic_range_compression_line ⇒ String rw

        Specify the absolute peak level for a signal with dynamic range compression.

      • #dynamic_range_compression_rf ⇒ String rw

        Specify how the service limits the audio dynamic range when compressing the audio.

      • #lfe_control ⇒ String rw

        When encoding 3/2 audio, controls whether the LFE channel is enabled.

      • #lfe_filter ⇒ String rw

        Applies a 120Hz lowpass filter to the LFE channel prior to encoding.

      • #lo_ro_center_mix_level ⇒ Float rw

        Specify a value for the following Dolby Digital Plus setting: Left only/Right only center mix (Lo/Ro center).

      • #lo_ro_surround_mix_level ⇒ Float rw

        Specify a value for the following Dolby Digital Plus setting: Left only/Right only (Lo/Ro surround).

      • #lt_rt_center_mix_level ⇒ Float rw

        Specify a value for the following Dolby Digital Plus setting: Left total/Right total center mix (Lt/Rt center).

      • #lt_rt_surround_mix_level ⇒ Float rw

        Specify a value for the following Dolby Digital Plus setting: Left total/Right total surround mix (Lt/Rt surround).

      • #metadata_control ⇒ String rw

        When set to FOLLOW_INPUT, encoder metadata will be sourced from the DD, DD+, or DolbyE decoder that supplied this audio data.

      • #passthrough_control ⇒ String rw

        When set to WHEN_POSSIBLE, input DD+ audio will be passed through if it is present on the input.

      • #phase_control ⇒ String rw

        Controls the amount of phase-shift applied to the surround channels.

      • #sample_rate ⇒ Integer rw

        This value is always 48000.

      • #stereo_downmix ⇒ String rw

        Choose how the service does stereo downmixing.

      • #surround_ex_mode ⇒ String rw

        When encoding 3/2 audio, sets whether an extra center back surround channel is matrix encoded into the left and right surround channels.

      • #surround_mode ⇒ String rw

        When encoding 2/0 audio, sets whether Dolby Surround is matrix encoded into the two channels.

    #lo_ro_center_mix_level  ⇒ Float  (rw)
  
      

      
  
      
    
      Specify a value for the following Dolby Digital Plus setting: Left
only/Right only center mix (Lo/Ro center). ::Aws::MediaConvert uses this
value for downmixing. How the service uses this value depends on the
value that you choose for Stereo downmix (Eac3StereoDownmix). Valid
values: 3.0, 1.5, 0.0, -1.5, -3.0, -4.5, -6.0, and -60. The value
-60 mutes the channel. This setting applies only if you keep the
default value of 3/2 - L, R, C, Ls, Rs (CODING_MODE_3_2) for the
setting Coding mode (Eac3CodingMode). If you choose a different
value for Coding mode, the service ignores Left only/Right only
center (loRoCenterMixLevel).
