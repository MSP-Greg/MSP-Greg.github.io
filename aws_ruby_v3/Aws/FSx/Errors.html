      Overview

      When FSx returns an error response, the Ruby SDK constructs and raises an error. These errors all extend Aws::FSx::Errors::ServiceError < ::Aws::Errors::ServiceError

      You can rescue all ::Aws::FSx errors using ServiceError:

      begin
  # do stuff
rescue Aws::FSx::Errors::ServiceError
  # rescues all FSx API errors
end

      Request Context

      ServiceError objects have a #context method that returns information about the request that generated the error. See ::Seahorse::Client::RequestContext for more information.
