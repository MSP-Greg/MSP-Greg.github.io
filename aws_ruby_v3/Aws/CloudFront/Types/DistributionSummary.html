      Instance Attribute Details

      #alias_icp_recordals ⇒ Array<Types::AliasICPRecordal> (rw)

      AWS services in China customers must file for an Internet Content Provider (ICP) recordal if they want to serve content publicly on an alternate domain name, also known as a CNAME, that they've added to ::Aws::CloudFront. AliasICPRecordal provides the ICP recordal status for CNAMEs associated with distributions.

      For more information about ICP recordals, see Signup, Accounts, and Credentials in Getting Started with AWS services in China.
