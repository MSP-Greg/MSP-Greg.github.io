      Overview

      This is AWS ::Aws::WAF Classic documentation. For more information, see AWS ::Aws::WAF Classic in the developer guide.

      For the latest version of AWS WAF, use the AWS WAFV2 API and see the AWS ::Aws::WAF Developer Guide. With the latest version, AWS WAF has a single set of endpoints for regional and global use.

      The response from a GetSampledRequests request includes a SampledHTTPRequests complex type that appears as SampledRequests in the response syntax. SampledHTTPRequests contains one SampledHTTPRequest object for each web request that is returned by GetSampledRequests.
