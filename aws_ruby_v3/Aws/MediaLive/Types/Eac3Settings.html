      Instance Attribute Summary

      • #attenuation_control ⇒ String rw

        When set to attenuate3Db, applies a 3 dB attenuation to the surround channels.

      • #bitrate ⇒ Float rw

        Average bitrate in bits/second.

      • #bitstream_mode ⇒ String rw

        Specifies the bitstream mode (bsmod) for the emitted E-AC-3 stream.

      • #coding_mode ⇒ String rw

        Dolby Digital Plus coding mode.

      • #dc_filter ⇒ String rw

        When set to enabled, activates a DC highpass filter for all input channels.

      • #dialnorm ⇒ Integer rw

        Sets the dialnorm for the output.

      • #drc_line ⇒ String rw

        Sets the Dolby dynamic range compression profile.

      • #drc_rf ⇒ String rw

        Sets the profile for heavy Dolby dynamic range compression, ensures that the instantaneous signal peaks do not exceed specified levels.

      • #lfe_control ⇒ String rw

        When encoding 3/2 audio, setting to lfe enables the LFE channel.

      • #lfe_filter ⇒ String rw

        When set to enabled, applies a 120Hz lowpass filter to the LFE channel prior to encoding.

      • #lo_ro_center_mix_level ⇒ Float rw

        Left only/Right only center mix level.

      • #lo_ro_surround_mix_level ⇒ Float rw

        Left only/Right only surround mix level.

      • #lt_rt_center_mix_level ⇒ Float rw

        Left total/Right total center mix level.

      • #lt_rt_surround_mix_level ⇒ Float rw

        Left total/Right total surround mix level.

      • #metadata_control ⇒ String rw

        When set to followInput, encoder metadata will be sourced from the DD, DD+, or DolbyE decoder that supplied this audio data.

      • #passthrough_control ⇒ String rw

        When set to whenPossible, input DD+ audio will be passed through if it is present on the input.

      • #phase_control ⇒ String rw

        When set to shift90Degrees, applies a 90-degree phase shift to the surround channels.

      • #stereo_downmix ⇒ String rw

        Stereo downmix preference.

      • #surround_ex_mode ⇒ String rw

        When encoding 3/2 audio, sets whether an extra center back surround channel is matrix encoded into the left and right surround channels.

      • #surround_mode ⇒ String rw

        When encoding 2/0 audio, sets whether Dolby Surround is matrix encoded into the two channels.

    #passthrough_control  ⇒ String  (rw)
  
      

      
  
      
    
      When set to whenPossible, input DD+ audio will be passed through if
it is present on the input. This detection is dynamic over the life
of the transcode. Inputs that alternate between DD+ and non-DD+
content will have a consistent DD+ output as the system alternates
between passthrough and encoding.
