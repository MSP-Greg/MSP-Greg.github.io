      Instance Attribute Details

      #error_clear_time_msec ⇒ Integer (rw)

      This clear time defines the requirement a recovered input must meet to be considered healthy. The input must have no failover conditions for this length of time. Enter a time in milliseconds. This value is particularly important if the input_preference for the failover pair is set to PRIMARY_INPUT_PREFERRED, because after this time, ::Aws::MediaLive will switch back to the primary input.
