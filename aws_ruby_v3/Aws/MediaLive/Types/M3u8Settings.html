      Instance Attribute Summary

      • #audio_frames_per_pes ⇒ Integer rw

        The number of audio frames to insert for each PES packet.

      • #audio_pids ⇒ String rw

        Packet Identifier (PID) of the elementary audio stream(s) in the transport stream.

      • #ecm_pid ⇒ String rw

        This parameter is unused and deprecated.

      • #nielsen_id_3_behavior ⇒ String rw

        If set to passthrough, Nielsen inaudible tones for media tracking will be detected in the input audio and an equivalent ID3 tag will be inserted in the output.

      • #pat_interval ⇒ Integer rw

        The number of milliseconds between instances of this table in the output transport stream.

      • #pcr_control ⇒ String rw

        When set to pcrEveryPesPacket, a Program Clock Reference value is inserted for every Packetized Elementary Stream (PES) header.

      • #pcr_period ⇒ Integer rw

        Maximum time in milliseconds between Program Clock References (PCRs) inserted into the transport stream.

      • #pcr_pid ⇒ String rw

        Packet Identifier (PID) of the Program Clock Reference (PCR) in the transport stream.

      • #pmt_interval ⇒ Integer rw

        The number of milliseconds between instances of this table in the output transport stream.

      • #pmt_pid ⇒ String rw

        Packet Identifier (PID) for the Program Map Table (PMT) in the transport stream.

      • #program_num ⇒ Integer rw

        The value of the program number field in the Program Map Table.

      • #scte_35_behavior ⇒ String rw

        If set to passthrough, passes any SCTE-35 signals from the input source to this output.

      • #scte_35_pid ⇒ String rw

        Packet Identifier (PID) of the SCTE-35 stream in the transport stream.

      • #timed_metadata_behavior ⇒ String rw

        When set to passthrough, timed metadata is passed through from input to output.

      • #timed_metadata_pid ⇒ String rw

        Packet Identifier (PID) of the timed metadata stream in the transport stream.

      • #transport_stream_id ⇒ Integer rw

        The value of the transport stream ID field in the Program Map Table.

      • #video_pid ⇒ String rw

        Packet Identifier (PID) of the elementary video stream in the transport stream.

inserted for every Packetized Elementary Stream (PES) header. This
parameter is effective only when the PCR PID is the same as the
