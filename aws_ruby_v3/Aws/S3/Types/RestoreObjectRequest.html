      #bucket ⇒ String (rw)

      The bucket name containing the object to restore.

      When using this action with an access point, you must direct requests to the access point hostname. The access point hostname takes the form AccessPointName-AccountId.s3-accesspoint.Region.amazonaws.com. When using this action with an access point through the AWS SDKs, you provide the access point ::Aws::ARN in place of the bucket name. For more information about access point ARNs, see Using Access Points in the Amazon Simple Storage Service Developer Guide.

      When using this action with Amazon S3 on ::Aws::Outposts, you must direct requests to the ::Aws::S3 on ::Aws::Outposts hostname. The S3 on ::Aws::Outposts hostname takes the form AccessPointName-AccountId.outpostID.s3-outposts.Region.amazonaws.com. When using this action using ::Aws::S3 on ::Aws::Outposts through the AWS SDKs, you provide the ::Aws::Outposts bucket ::Aws::ARN in place of the bucket name. For more information about ::Aws::S3 on ::Aws::Outposts ARNs, see Using ::Aws::S3 on Outposts in the Amazon Simple Storage Service Developer Guide.
