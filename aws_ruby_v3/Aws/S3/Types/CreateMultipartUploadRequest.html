    #bucket_key_enabled  ⇒ Boolean  (rw)
  
      

      
  
      
    
      Specifies whether Amazon S3 should use an ::Aws::S3 Bucket Key for object
encryption with server-side encryption using AWS KMS (SSE-KMS).
Setting this header to true causes Amazon S3 to use an ::Aws::S3 Bucket
Key for object encryption with SSE-KMS.
      

      Specifying this header with an object action doesn’t affect
bucket-level settings for ::Aws::S3 Bucket Key.
