      Note:

      When making an API call, you may pass Source data as a hash:

      {
  type: "git", # accepts git, svn, archive, s3
  url: "String",
  username: "String",
  password: "String",
  ssh_key: "String",
  revision: "String",
}

      Contains the information required to retrieve an app or cookbook from a repository. For more information, see Creating Apps or Custom Recipes and Cookbooks.

      Instance Attribute Details

      #password ⇒ String (rw)

      When included in a request, the parameter depends on the repository type.

      • For Amazon ::Aws::S3 bundles, set Password to the appropriate ::Aws::IAM secret access key.

      • For HTTP bundles and Subversion repositories, set Password to the password.

      For more information on how to safely handle ::Aws::IAM credentials, see https://docs.aws.amazon.com/general/latest/gr/aws-access-keys-best-practices.html.
