      Overview

      When SSM returns an error response, the Ruby SDK constructs and raises an error. These errors all extend Aws::SSM::Errors::ServiceError < ::Aws::Errors::ServiceError

      You can rescue all ::Aws::SSM errors using ServiceError:

      begin
  # do stuff
rescue Aws::SSM::Errors::ServiceError
  # rescues all SSM API errors
end

      Request Context

      ServiceError objects have a #context method that returns information about the request that generated the error. See ::Seahorse::Client::RequestContext for more information.
