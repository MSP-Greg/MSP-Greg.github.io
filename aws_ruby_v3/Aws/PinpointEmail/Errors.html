      Overview

      When PinpointEmail returns an error response, the Ruby SDK constructs and raises an error. These errors all extend Aws::PinpointEmail::Errors::ServiceError < ::Aws::Errors::ServiceError

      You can rescue all ::Aws::PinpointEmail errors using ServiceError:

      begin
  # do stuff
rescue Aws::PinpointEmail::Errors::ServiceError
  # rescues all PinpointEmail API errors
end

      Request Context

      ServiceError objects have a #context method that returns information about the request that generated the error. See ::Seahorse::Client::RequestContext for more information.
