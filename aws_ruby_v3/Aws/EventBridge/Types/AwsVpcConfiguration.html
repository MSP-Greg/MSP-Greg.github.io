      Overview

      Note:

      When making an API call, you may pass AwsVpcConfiguration data as a hash:

      {
  subnets: ["String"], # required
  security_groups: ["String"],
  assign_public_ip: "ENABLED", # accepts ENABLED, DISABLED
}

      This structure specifies the VPC subnets and security groups for the task, and whether a public IP address is to be used. This structure is relevant only for ::Aws::ECS tasks that use the awsvpc network mode.
