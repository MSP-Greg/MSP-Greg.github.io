    #direct_connect_gateway_state  ⇒ String  (rw)
  
      

      
  
      
    
      The state of the Direct Connect gateway. The following are the
possible values:
      

        

      • 

        pending: The initial state after calling
CreateDirectConnectGateway.
        

        • 

      • 

        available: The Direct ::Aws::Connect gateway is ready for use.
        

        • 

      • 

        deleting: The initial state after calling
DeleteDirectConnectGateway.
        

        • 

      • 

        deleted: The Direct ::Aws::Connect gateway is deleted and cannot pass
traffic.
