      Instance Attribute Details

      #hierarchy_id ⇒ String (rw)

      The ID of the hierarchy to query for the property ID. You can use the hierarchy's name instead of the hierarchy's ID.

      You use a hierarchy ID instead of a model ID because you can have several hierarchies using the same model and therefore the same propertyId. For example, you might have separately grouped assets that come from the same asset model. For more information, see Asset hierarchies in the AWS ::Aws::IoT SiteWise User Guide.
