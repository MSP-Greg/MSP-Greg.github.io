      Overview

      Contains the details for an Amazon RDS DB cluster snapshot

      This data type is used as a response element in the DescribeDBClusterSnapshots action.

      See Also:

        Specifies the allocated storage size in gibibytes (GiB).

      • #availability_zones ⇒ Array<String> rw

        Provides the list of Availability Zones (AZs) where instances in the DB cluster snapshot can be restored.

      • #cluster_create_time ⇒ Time rw

        Specifies the time when the DB cluster was created, in Universal Coordinated Time (UTC).

      • #db_cluster_identifier ⇒ String rw

        Specifies the DB cluster identifier of the DB cluster that this DB cluster snapshot was created from.

        The Amazon ::Aws::RDS::Resource Name (ARN) for the DB cluster snapshot.

      • #db_cluster_snapshot_identifier ⇒ String rw

        Specifies the identifier for the DB cluster snapshot.

      • #engine ⇒ String rw

        Specifies the name of the database engine for this DB cluster snapshot.

      • #engine_mode ⇒ String rw

        Provides the engine mode of the database engine for this DB cluster snapshot.

      • #engine_version ⇒ String rw

        Provides the version of the database engine for this DB cluster snapshot.

      • #iam_database_authentication_enabled ⇒ Boolean rw

        True if mapping of AWS Identity and Access Management (IAM) accounts to database accounts is enabled, and otherwise false.

      • #kms_key_id ⇒ String rw

        If StorageEncrypted is true, the AWS KMS key identifier for the encrypted DB cluster snapshot.

      • #license_model ⇒ String rw

        Provides the license model information for this DB cluster snapshot.

      • #master_username ⇒ String rw

        Provides the master username for this DB cluster snapshot.

      • #percent_progress ⇒ Integer rw

        Specifies the percentage of the estimated data that has been transferred.

      • #port ⇒ Integer rw

        Specifies the port that the DB cluster was listening on at the time of the snapshot.

      • #snapshot_create_time ⇒ Time rw

        Provides the time when the snapshot was taken, in Universal Coordinated Time (UTC).

      • #snapshot_type ⇒ String rw

        Provides the type of the DB cluster snapshot.

      • #source_db_cluster_snapshot_arn ⇒ String rw

        If the DB cluster snapshot was copied from a source DB cluster snapshot, the Amazon Resource Name (ARN) for the source DB cluster snapshot, otherwise, a null value.

      • #status ⇒ String rw

        Specifies the status of this DB cluster snapshot.

      • #storage_encrypted ⇒ Boolean rw

        Specifies whether the DB cluster snapshot is encrypted.

      • #tag_list ⇒ Array<Types::Tag> rw

        A list of tags.

      • #vpc_id ⇒ String rw

        Provides the VPC ID associated with the DB cluster snapshot.
