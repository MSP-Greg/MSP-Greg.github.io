      Overview

      This documentation is for version 1 of the Amazon Kinesis Data Analytics API, which only supports SQL applications. Version 2 of the API supports SQL and Java applications. For more information about version 2, see Amazon ::Aws::Kinesis Data Analytics API V2 Documentation.

      Provides a description of the application, including the application Amazon Resource Name (ARN), status, latest version, and input and output configuration.
