      Instance Attribute Summary

      • #input_id ⇒ String rw

        Input ID of the application input to be updated.

      • #input_parallelism_update ⇒ Types::InputParallelismUpdate rw

        Describes the parallelism updates (the number in-application streams Amazon Kinesis Analytics creates for the specific streaming source).

      • #input_processing_configuration_update ⇒ Types::InputProcessingConfigurationUpdate rw

        Describes updates for an input processing configuration.

      • #input_schema_update ⇒ Types::InputSchemaUpdate rw

        Describes the data format on the streaming source, and how record elements on the streaming source map to columns of the in-application stream that is created.

      • #kinesis_firehose_input_update ⇒ Types::KinesisFirehoseInputUpdate rw

        If an Amazon Kinesis ::Aws::Firehose delivery stream is the streaming source to be updated, provides an updated stream ::Aws::ARN and ::Aws::IAM role ::Aws::ARN.

      • #kinesis_streams_input_update ⇒ Types::KinesisStreamsInputUpdate rw

        If an Amazon Kinesis stream is the streaming source to be updated, provides an updated stream Amazon Resource Name (ARN) and ::Aws::IAM role ::Aws::ARN.

      • #name_prefix_update ⇒ String rw

        Name prefix for in-application streams that Amazon Kinesis Analytics creates for the specific streaming source.
