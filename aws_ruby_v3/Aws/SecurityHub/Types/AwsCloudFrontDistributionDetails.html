      • #default_root_object ⇒ String rw

        The object that ::Aws::CloudFront sends in response to requests from the origin (for example, index.html) when a viewer requests the root URL for the distribution (http://www.example.com) instead of an object in your distribution (http://www.example.com/product-description.html).

      • #status ⇒ String rw

        Indicates the current status of the distribution.

        A unique identifier that specifies the AWS WAF web ACL, if any, to associate with this distribution.
