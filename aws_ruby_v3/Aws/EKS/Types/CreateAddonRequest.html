    #service_account_role_arn  ⇒ String  (rw)
  
      

      
  
      
    
      The Amazon ::Aws::EKS::Resource Name (ARN) of an existing ::Aws::IAM role to bind to
the add-on's service account. The role must be assigned the ::Aws::IAM
permissions required by the add-on. If you don't specify an
existing ::Aws::IAM role, then the add-on uses the permissions assigned to
the node ::Aws::IAM role. For more information, see Amazon ::Aws::EKS node ::Aws::IAM
role in the Amazon ::Aws::EKS User Guide.
      

       To specify an existing ::Aws::IAM role, you must have an ::Aws::IAM OpenID ::Aws::Connect
(OIDC) provider created for your cluster. For more information, see
Enabling ::Aws::IAM roles for service accounts on your cluster in the
Amazon ::Aws::EKS User Guide.
