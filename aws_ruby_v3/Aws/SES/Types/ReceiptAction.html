      An action that Amazon SES can take when it receives an email on behalf of one or more email addresses or domains that you own. An instance of this data type can represent only one action.

      For information about setting up receipt rules, see the Amazon ::Aws::SES Developer Guide.

      Instance Attribute Summary

      • #add_header_action ⇒ Types::AddHeaderAction rw

        Adds a header to the received email.

      • #bounce_action ⇒ Types::BounceAction rw

        Rejects the received email by returning a bounce response to the sender and, optionally, publishes a notification to Amazon Simple Notification Service (Amazon ::Aws::SNS).

      • #lambda_action ⇒ Types::LambdaAction rw

        Calls an AWS Lambda function, and optionally, publishes a notification to Amazon SNS.

      • #s3_action ⇒ Types::S3Action rw

        Saves the received message to an Amazon Simple Storage Service (Amazon ::Aws::S3) bucket and, optionally, publishes a notification to Amazon SNS.

      • #sns_action ⇒ Types::SNSAction rw

        Publishes the email content within a notification to Amazon SNS.

      • #stop_action ⇒ Types::StopAction rw

        Terminates the evaluation of the receipt rule set and optionally publishes a notification to Amazon SNS.

      • #workmail_action ⇒ Types::WorkmailAction rw

        Calls Amazon ::Aws::WorkMail and, optionally, publishes a notification to Amazon Amazon ::Aws::SNS.
