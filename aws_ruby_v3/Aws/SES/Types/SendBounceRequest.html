    #bounce_sender_arn  ⇒ String  (rw)
  
      

      
  
      
    
      This parameter is used only for sending authorization. It is the ::Aws::ARN
of the identity that is associated with the sending authorization
policy that permits you to use the address in the "From" header of
the bounce. For more information about sending authorization, see
the Amazon ::Aws::SES Developer Guide.
