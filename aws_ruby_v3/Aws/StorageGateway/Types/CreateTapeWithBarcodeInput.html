    #pool_id  ⇒ String  (rw)
  
      

      
  
      
    
      The ID of the pool that you want to add your tape to for archiving.
The tape in this pool is archived in the ::Aws::S3 storage class that is
associated with the pool. When you use your backup application to
eject the tape, the tape is archived directly into the storage class
(S3 ::Aws::Glacier or ::Aws::S3 Deep Archive) that corresponds to the pool.
      

      Valid Values: GLACIER | DEEP_ARCHIVE
