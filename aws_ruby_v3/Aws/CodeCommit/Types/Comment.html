      • #author_arn ⇒ String rw

        The Amazon ::Aws::CodeCommit::Resource Name (ARN) of the person who posted the comment.

      • #caller_reactions ⇒ Array<String> rw

        The emoji reactions to a comment, if any, submitted by the user whose credentials are associated with the call to the API.

      • #client_request_token ⇒ String rw

        A unique, client-generated idempotency token that, when provided in a request, ensures the request cannot be repeated with a changed parameter.

      • #comment_id ⇒ String rw

        The system-generated comment ID.

      • #content ⇒ String rw

        The content of the comment.

      • #creation_date ⇒ Time rw

        The date and time the comment was created, in timestamp format.

      • #deleted ⇒ Boolean rw

        A Boolean value indicating whether the comment has been deleted.

      • #in_reply_to ⇒ String rw

        The ID of the comment for which this comment is a reply, if any.

      • #last_modified_date ⇒ Time rw

        The date and time the comment was most recently modified, in timestamp format.

      • #reaction_counts ⇒ Hash<String,Integer> rw

        A string to integer map that represents the number of individual users who have responded to a comment with the specified reactions.
