    #max_queue_size  ⇒ Integer  (rw)
  
      

      
  
      
    
      The maximum size of the message queue for the ::Aws::Lambda function
component. The AWS ::Aws::IoT Greengrass core stores messages in a FIFO
(first-in-first-out) queue until it can run the ::Aws::Lambda function to
consume each message.

    #pinned  ⇒ Boolean  (rw)
  
      

      
  
      
    
      Whether or not the ::Aws::Lambda function is pinned, or long-lived.
      

        

      • 

        A pinned ::Aws::Lambda function starts when AWS IoT ::Aws::Greengrass starts and
keeps running in its own container.
        

        • 

      • 

        A non-pinned ::Aws::Lambda function starts only when it receives a work
item and exists after it idles for maxIdleTimeInSeconds. If the
function has multiple work items, the AWS IoT Greengrass Core
software creates multiple instances of the function.
        

        • 

      

      Default: true
