      Instance Attribute Summary

      • #amazon_id ⇒ String rw

        The device type ID for your AVS device generated by Amazon when the OEM creates a new product on Amazon's Developer Console.

      • #client_id ⇒ String rw

        The client ID of the OEM used for code-based linking authorization on an AVS device.

      • #device_serial_number ⇒ String rw

        The key generated by the OEM that uniquely identifies a specified instance of your AVS device.

      • #product_id ⇒ String rw

        The product ID used to identify your AVS device during authorization.

      • #room_arn ⇒ String rw

        The Amazon ::Aws::AlexaForBusiness::Resource Name (ARN) of the room with which to associate your AVS device.

      • #tags ⇒ Array<Types::Tag> rw

        The tags to be added to the specified resource.

      • #user_code ⇒ String rw

        The code that is obtained after your AVS device has made a POST request to LWA as a part of the Device Authorization Request component of the OAuth code-based linking specification.
