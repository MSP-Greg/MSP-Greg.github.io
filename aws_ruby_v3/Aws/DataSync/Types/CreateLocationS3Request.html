    #s3_storage_class  ⇒ String  (rw)
  
      

      
  
      
    
      The Amazon ::Aws::S3 storage class that you want to store your files in
when this location is used as a task destination. For buckets in AWS
Regions, the storage class defaults to Standard. For buckets on AWS
::Aws::Outposts, the storage class defaults to AWS S3 ::Aws::Outposts.
      

      For more information about ::Aws::S3 storage classes, see Amazon ::Aws::S3
Storage Classes. Some storage classes have behaviors that can
affect your ::Aws::S3 storage cost. For detailed information, see
using-storage-classes.
