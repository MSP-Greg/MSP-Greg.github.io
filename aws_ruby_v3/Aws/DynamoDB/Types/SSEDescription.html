      Instance Attribute Details

      #inaccessible_encryption_date_time ⇒ Time (rw)

      Indicates the time, in UNIX epoch date format, when ::Aws::DynamoDB detected that the table's AWS KMS key was inaccessible. This attribute will automatically be cleared when ::Aws::DynamoDB detects that the table's AWS KMS key is accessible again. ::Aws::DynamoDB will initiate the table archival process when table's AWS KMS key remains inaccessible for more than seven days from this date.
