    #kms_master_key_id  ⇒ String  (rw)
  
      

      
  
      
    
      The AWS ::Aws::KMS customer master key (CMK) that should be used for AWS
::Aws::KMS encryption in the new replica. To specify a CMK, use its key ID,
Amazon Resource Name (ARN), alias name, or alias ::Aws::ARN. Note that you
should only provide this parameter if the key is different from the
default ::Aws::DynamoDB KMS master key alias/aws/dynamodb.
