    #enable_internet_access  ⇒ Boolean  (rw)
  
      

      
  
      
    
      Specifies whether to automatically assign an Elastic public IP
address to ::Aws::WorkSpaces in this directory by default. If enabled, the
Elastic public IP address allows outbound internet access from your
::Aws::WorkSpaces when you’re using an internet gateway in the Amazon VPC
in which your ::Aws::WorkSpaces are located. If you're using a Network
Address Translation (NAT) gateway for outbound internet access from
your VPC, or if your ::Aws::WorkSpaces are in public subnets and you
manually assign them Elastic IP addresses, you should disable this
setting. This setting applies to new ::Aws::WorkSpaces that you launch or
to existing ::Aws::WorkSpaces that you rebuild. For more information, see 
Configure a VPC for Amazon WorkSpaces.
