      Instance Attribute Details

      #directory_id ⇒ String (rw)

      The identifier of the directory. You cannot register a directory if it does not have a status of Active. If the directory does not have a status of Active, you will receive an InvalidResourceStateException error. If you have already registered the maximum number of directories that you can register with Amazon ::Aws::WorkSpaces, you will receive a ResourceLimitExceededException error. Deregister directories that you are not using for ::Aws::WorkSpaces, and try again.
