      • #application_id ⇒ String rw

        The unique identifier for the application that the campaign applies to.

      • #campaign_id ⇒ String rw

        The unique identifier for the campaign that the activity applies to.

      • #end ⇒ String rw

        The actual time, in ISO 8601 format, when the activity was marked CANCELLED or COMPLETED.

      • #id ⇒ String rw

        The unique identifier for the activity.

      • #result ⇒ String rw

        Specifies whether the activity succeeded.

      • #scheduled_start ⇒ String rw

        The scheduled start time, in ISO 8601 format, for the activity.

      • #start ⇒ String rw

        The actual start time, in ISO 8601 format, of the activity.

      • #state ⇒ String rw

        The current status of the activity.

      • #successful_endpoint_count ⇒ Integer rw

        The total number of endpoints that the campaign successfully delivered messages to.

      • #timezones_completed_count ⇒ Integer rw

        The total number of time zones that were completed.

      • #timezones_total_count ⇒ Integer rw

        The total number of unique time zones that are in the segment for the campaign.

      • #total_endpoint_count ⇒ Integer rw

        The total number of endpoints that the campaign attempted to deliver messages to.

      • #treatment_id ⇒ String rw

        The unique identifier for the campaign treatment that the activity applies to.
