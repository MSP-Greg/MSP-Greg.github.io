      Instance Attribute Details

      #name ⇒ String (rw)

      The Amazon ::Aws::ECS resource name for which to modify the account setting. If serviceLongArnFormat is specified, the ::Aws::ARN for your Amazon ECS services is affected. If taskLongArnFormat is specified, the ::Aws::ARN and resource ID for your Amazon ECS tasks is affected. If containerInstanceLongArnFormat is specified, the ::Aws::ARN and resource ID for your Amazon ECS container instances is affected. If awsvpcTrunking is specified, the elastic network interface (ENI) limit for your Amazon ECS container instances is affected. If containerInsights is specified, the default setting for ::Aws::CloudWatch Container Insights for your clusters is affected.

    #principal_arn  ⇒ String  (rw)
  
      

      
  
      
    
      The ARN of the principal, which can be an ::Aws::IAM user, ::Aws::IAM role, or the
root user. If you specify the root user, it modifies the account
setting for all ::Aws::IAM users, ::Aws::IAM roles, and the root user of the
account unless an ::Aws::IAM user or role explicitly overrides these
settings. If this field is omitted, the setting is changed only for
the authenticated user.
