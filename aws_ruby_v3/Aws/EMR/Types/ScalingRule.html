      A scale-in or scale-out rule that defines scaling activity, including the ::Aws::CloudWatch metric alarm that triggers activity, how ::Aws::EC2 instances are added or removed, and the periodicity of adjustments. The automatic scaling policy for an instance group can comprise one or more automatic scaling rules.
