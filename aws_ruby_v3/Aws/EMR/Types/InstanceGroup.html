      Overview

      This entity represents an instance group, which is a group of instances that have common purpose. For example, CORE instance group is used for HDFS.

      #auto_scaling_policy ⇒ Types::AutoScalingPolicyDescription (rw)

      An automatic scaling policy for a core instance group or task instance group in an Amazon EMR cluster. The automatic scaling policy defines how an instance group dynamically adds and terminates ::Aws::EC2 instances in response to the value of a ::Aws::CloudWatch metric. See PutAutoScalingPolicy.
