      #account_recovery_setting ⇒ Types::AccountRecoverySettingType (rw)

      Use this setting to define which verified available method a user can use to recover their password when they call ForgotPassword. It allows you to define a preferred method when a user has more than one method available. With this setting, ::Aws::SMS does not qualify for a valid password recovery mechanism if the user also has ::Aws::SMS MFA enabled. In the absence of this setting, Cognito uses the legacy behavior to determine the recovery method where ::Aws::SMS is preferred over email.
