      Instance Attribute Summary

      • #allow_interrupt ⇒ Boolean rw

        Indicates that the user can interrupt the response by speaking while the message is being played.

      • #frequency_in_seconds ⇒ Integer rw

        How often a message should be sent to the user.

      • #message_groups ⇒ Array<Types::MessageGroup> rw

        One or more message groups, each containing one or more messages, that define the prompts that Amazon Lex sends to the user.

      • #timeout_in_seconds ⇒ Integer rw

        If Amazon ::Aws::Lex waits longer than this length of time for a response, it will stop sending messages.
