      Instance Attribute Details

      #active_tracing ⇒ Boolean (rw)

      Specifies whether this canary is to use active AWS X-Ray tracing when it runs. Active tracing enables this canary run to be displayed in the ServiceLens and X-Ray service maps even if the canary does not hit an endpoint that has X-ray tracing enabled. Using X-Ray tracing incurs charges. For more information, see Canaries and X-Ray tracing.

      You can enable active tracing only for canaries that use version syn-nodejs-2.0 or later for their canary runtime.

    #environment_variables  ⇒ Hash<String,String>  (rw)
  
      

      
  
      
    
      Specifies the keys and values to use for any environment variables
used in the canary script. Use the following format:
      

      \{ "key1" : "value1", "key2" : "value2", ...\}
      

      Keys must start with a letter and be at least two characters. The
total size of your environment variables cannot exceed 4 KB. You
can't specify any ::Aws::Lambda reserved environment variables as the keys
for your environment variables. For more information about reserved
keys, see  Runtime environment variables.

      How long the canary is allowed to run before it must stop. You
