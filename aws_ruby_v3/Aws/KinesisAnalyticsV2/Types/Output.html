      Overview

      Note:

      When making an API call, you may pass Output data as a hash:

      {
  name: "InAppStreamName", # required
  kinesis_streams_output: {
    resource_arn: "ResourceARN", # required
  },
  kinesis_firehose_output: {
    resource_arn: "ResourceARN", # required
  },
  lambda_output: {
    resource_arn: "ResourceARN", # required
  },
  destination_schema: { # required
    record_format_type: "JSON", # required, accepts JSON, CSV
  },
}

      Describes a SQL-based ::Aws::Kinesis Data Analytics application's output configuration, in which you identify an in-application stream and a destination where you want the in-application stream data to be written. The destination can be a ::Aws::Kinesis data stream or a ::Aws::Kinesis Data Firehose delivery stream.
