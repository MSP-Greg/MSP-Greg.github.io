      The date of the automatic snapshot to use for the new instance. Use
the get auto snapshots operation to identify the dates of the
available automatic snapshots.
      

      Constraints:
      

        

      • 

        Must be specified in YYYY-MM-DD format.
        

        •

        This parameter cannot be defined together with the use latest restorable auto snapshot parameter. The restore date and use latest restorable auto snapshot parameters are mutually

        • 

      •

Guide.
