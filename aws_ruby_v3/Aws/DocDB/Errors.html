      Overview

      When DocDB returns an error response, the Ruby SDK constructs and raises an error. These errors all extend Aws::DocDB::Errors::ServiceError < ::Aws::Errors::ServiceError

      You can rescue all ::Aws::DocDB errors using ServiceError:

      begin
  # do stuff
rescue Aws::DocDB::Errors::ServiceError
  # rescues all DocDB API errors
end

      Request Context

      ServiceError objects have a #context method that returns information about the request that generated the error. See ::Seahorse::Client::RequestContext for more information.
