      Instance Attribute Summary

      • #availability_zones ⇒ Array<String> rw

        Provides the list of Amazon EC2 Availability Zones that instances in the cluster snapshot can be restored in.

      • #cluster_create_time ⇒ Time rw

        Specifies the time when the cluster was created, in Universal Coordinated Time (UTC).

      • #db_cluster_identifier ⇒ String rw

        Specifies the cluster identifier of the cluster that this cluster snapshot was created from.

      • #db_cluster_snapshot_arn ⇒ String rw

        The Amazon ::Aws::DocDB::Resource Name (ARN) for the cluster snapshot.

      • #db_cluster_snapshot_identifier ⇒ String rw

        Specifies the identifier for the cluster snapshot.

      • #engine ⇒ String rw

        Specifies the name of the database engine.

      • #engine_version ⇒ String rw

        Provides the version of the database engine for this cluster snapshot.

      • #kms_key_id ⇒ String rw

        If StorageEncrypted is true, the AWS KMS key identifier for the encrypted cluster snapshot.

      • #master_username ⇒ String rw

        Provides the master user name for the cluster snapshot.

      • #percent_progress ⇒ Integer rw

        Specifies the percentage of the estimated data that has been transferred.

      • #port ⇒ Integer rw

        Specifies the port that the cluster was listening on at the time of the snapshot.

      • #snapshot_create_time ⇒ Time rw

        Provides the time when the snapshot was taken, in UTC.

      • #snapshot_type ⇒ String rw

        Provides the type of the cluster snapshot.

      • #source_db_cluster_snapshot_arn ⇒ String rw

        If the cluster snapshot was copied from a source cluster snapshot, the ::Aws::ARN for the source cluster snapshot; otherwise, a null value.

      • #status ⇒ String rw

        Specifies the status of this cluster snapshot.

      • #storage_encrypted ⇒ Boolean rw

        Specifies whether the cluster snapshot is encrypted.

      • #vpc_id ⇒ String rw

        Provides the virtual private cloud (VPC) ID that is associated with the cluster snapshot.
