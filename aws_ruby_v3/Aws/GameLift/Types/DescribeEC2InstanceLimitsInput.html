      Name of an ::Aws::EC2 instance type that is supported in Amazon GameLift. A fleet instance type determines the computing resources of each instance in the fleet, including CPU, memory, storage, and networking capacity. Amazon GameLift supports the following ::Aws::EC2 instance types. See Amazon ::Aws::EC2 Instance Types for detailed descriptions. Leave this parameter blank to retrieve limits for all types.
