      This data type is used with the Amazon GameLift FleetIQ and game server groups.

      An allowed instance type for a GameServerGroup. All game server groups must have at least two instance types defined for it. ::Aws::GameLift FleetIQ periodically evaluates each defined instance type for viability. It then updates the Auto Scaling group with the list of viable instance types.

    #weighted_capacity  ⇒ String  (rw)
  
      

      
  
      
    
      Instance weighting that indicates how much this instance type
contributes to the total capacity of a game server group. Instance
weights are used by ::Aws::GameLift FleetIQ to calculate the instance
type's cost per unit hour and better identify the most
cost-effective options. For detailed information on weighting
instance capacity, see Instance Weighting in the Amazon ::Aws::EC2
Auto Scaling User Guide. Default value is "1".
