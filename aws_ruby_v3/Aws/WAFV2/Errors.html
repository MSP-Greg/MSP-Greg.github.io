      Overview

      When WAFV2 returns an error response, the Ruby SDK constructs and raises an error. These errors all extend Aws::WAFV2::Errors::ServiceError < ::Aws::Errors::ServiceError

      You can rescue all ::Aws::WAFV2 errors using ServiceError:

      begin
  # do stuff
rescue Aws::WAFV2::Errors::ServiceError
  # rescues all WAFV2 API errors
end

      Request Context

      ServiceError objects have a #context method that returns information about the request that generated the error. See ::Seahorse::Client::RequestContext for more information.
