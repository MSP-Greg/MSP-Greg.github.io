      Specifies whether this is for an AWS CloudFront distribution or for
a regional application. A regional application can be an Application
Load Balancer (ALB), an API Gateway REST API, or an ::Aws::AppSync GraphQL
API.
      

      To work with ::Aws::CloudFront, you must also specify the Region US East
(N. Virginia) as follows:
      

        

      • 

        CLI - Specify the Region when you use the ::Aws::CloudFront scope:
--scope=CLOUDFRONT --region=us-east-1.
        

        • 

      • 

        API and SDKs - For all calls, use the Region endpoint us-east-1.
