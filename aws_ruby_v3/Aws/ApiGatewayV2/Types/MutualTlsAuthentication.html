      Instance Attribute Details

      #truststore_uri ⇒ String (rw)

      An Amazon ::Aws::S3 URL that specifies the truststore for mutual TLS authentication, for example, s3://bucket-name/key-name . The truststore can contain certificates from public or private certificate authorities. To update the truststore, upload a new version to ::Aws::S3, and then update your custom domain name to use the new version. To update the truststore, you must have permissions to access the ::Aws::S3 object.

truststore. Invalid certificates produce warnings. Mutual TLS is
still enabled, but some clients might not be able to access your
