      • #billing_view_arn ⇒ String rw

        The Amazon resource name of the billing view.

      • #compression ⇒ String rw

        The compression format that AWS uses for the report.

      • #format ⇒ String rw

        The format that AWS saves the report in.

      • #refresh_closed_reports ⇒ Boolean rw

        Whether you want Amazon Web Services to update your reports after they have been finalized if Amazon Web Services detects charges related to previous months.

      • #report_name ⇒ String rw

        The name of the report that you want to create.

      • #report_versioning ⇒ String rw

        Whether you want Amazon Web Services to overwrite the previous version of each report or to deliver the report in addition to the previous versions.

      • #s3_bucket ⇒ String rw

        The S3 bucket where AWS delivers the report.

      • #s3_prefix ⇒ String rw

        The prefix that AWS adds to the report name when AWS delivers the report.

      • #s3_region ⇒ String rw

        The region of the ::Aws::S3 bucket that AWS delivers the report into.

      • #time_unit ⇒ String rw

        The length of time covered by the report.
