      Instance Attribute Summary

      • #description ⇒ String rw

        Description specified by the CreateSnapshotRequest that has been applied to all snapshots.

      • #encrypted ⇒ Boolean rw

        Indicates whether the snapshot is encrypted.

      • #outpost_arn ⇒ String rw

        The ARN of the AWS Outpost on which the snapshot is stored.

      • #owner_id ⇒ String rw

        Account id used when creating this snapshot.

      • #progress ⇒ String rw

        Progress this snapshot has made towards completing.

      • #snapshot_id ⇒ String rw

        Snapshot id that can be used to describe this snapshot.

      • #start_time ⇒ Time rw

        Time this snapshot was started.

      • #state ⇒ String rw

        Current state of the snapshot.

      • #tags ⇒ Array<Types::Tag> rw

        Tags associated with this snapshot.

      • #volume_id ⇒ String rw

        Source volume from which this snapshot was created.

      • #volume_size ⇒ Integer rw

        Size of the volume from which this snapshot was created.
