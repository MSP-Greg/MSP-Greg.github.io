      Instance Attribute Details

      #allocation_strategy ⇒ String (rw)

      The order of the launch template overrides to use in fulfilling On-Demand capacity. If you specify lowest-price, ::Aws::EC2 Fleet uses price to determine the order, launching the lowest price first. If you specify prioritized, ::Aws::EC2 Fleet uses the priority that you assigned to each launch template override, launching the highest priority first. If you do not specify a value, ::Aws::EC2 Fleet defaults to lowest-price.
