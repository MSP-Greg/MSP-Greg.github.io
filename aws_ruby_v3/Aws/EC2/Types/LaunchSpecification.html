    #ebs_optimized  ⇒ Boolean  (rw)
  
      

      
  
      
    
      Indicates whether the instance is optimized for ::Aws::EBS I/O. This
optimization provides dedicated throughput to Amazon EBS and an
optimized configuration stack to provide optimal ::Aws::EBS I/O
performance. This optimization isn't available with all instance
types. Additional usage charges apply when using an ::Aws::EBS Optimized
instance.
      

      Default: false
