    #obfuscation_setting  ⇒ String  (rw)
  
      

      
  
      
    
      Determines whether a slot is obfuscated in conversation logs and
stored utterances. When you obfuscate a slot, the value is replaced
by the slot name in curly braces ({\}). For example, if the slot
name is "full_name", obfuscated values are replaced with
"{full_name\}". For more information, see  Slot Obfuscation
.

    #priority  ⇒ Integer  (rw)
  
      

      
  
      
    
      Directs Amazon ::Aws::Lex the order in which to elicit this slot value from
the user. For example, if the intent has two slots with priorities 1
and 2, AWS Amazon ::Aws::Lex first elicits a value for the slot with
priority 1.
      

      If multiple slots share the same priority, the order in which Amazon
::Aws::Lex elicits values is arbitrary.
