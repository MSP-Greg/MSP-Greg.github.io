      Tools

      This documentation describes major tools related to RBS. They are listed alphabetically.

      Type Checkers

      • Steep is a static type checker based on RBS. You can use it as a language server integrated with an editor or IDE and run it from the command line.
      • TypeProf

      Editor integrations

      • Emacs: rbs-mode
      • RubyMine: It supports RBS by default
      • Sublime Text: sublime-rbs-plugin
      • Vim: rbs.vim
      • Visual Studio Code: vscode-rbs-syntax
      • NeoVim: tree-sitter-rbs
