      Overview

      This represents the result of a call to parse or parse_file. It contains the AST, any comments that were encounters, and any errors that were encountered.

def mark_newlines!
  value = self.value
  raise "This method should only be called on a parse result that contains a node" unless Node === value
  value.accept(Newlines.new(Array.new(1 + source.offsets.size, false))) # steep:ignore
end
