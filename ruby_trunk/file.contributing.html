      Contributing to Ruby

      Ruby Issues

      To report an issue in the Ruby core:

      • Report issues.

      Ruby Core

      To contribute to the Ruby core functionality, you'll need initially to:

      • Build Ruby on your system.
      • Test Ruby, to make sure the build is correct.

      Then:

      • Make changes to Ruby.

      And possibly:

      • Benchmark Ruby.

      To contribute to the Ruby core documentation, see:
