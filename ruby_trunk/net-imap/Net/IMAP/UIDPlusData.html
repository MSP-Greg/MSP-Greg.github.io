      Overview

      UIDPlusData represents the ResponseCode#data that accompanies the APPENDUID and COPYUID response codes.

      See [[UIDPLUS].

      Capability requirement

      The UIDPLUS capability must be supported. A server that supports UIDPLUS should send a UIDPlusData object inside every TaggedResponse returned by the append, copy, move, uid copy, and uid move commands—unless the destination mailbox reports UIDNOTSTICKY.
