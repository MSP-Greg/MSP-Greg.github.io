      You're welcome on our mailing list for any discussions around RubyInstaller. Bug reports and pull requests are also very welcome. We accept contributions covered under the MIT License.

      RubyInstaller consists of several single projects:

      • Main repository with build and test scripts and rubyinstaller runtime library: https://github.com/oneclick/rubyinstaller2
      • MSYS2/MINGW based repository for dependent binary packages: https://github.com/oneclick/rubyinstaller2-packages
      • The rubyinstaller.org web site: https://github.com/oneclick/rubyinstaller.org-website
